PROCEDIMENTO OPERACIONAL PADRÃO

Título: Montagem do Pack de Baterias (Conexão de Células)

Código: POP-003 Revisão: 01
Data de Emissão: 12/10/2025 Próxima Revisão: 12/10/2026
Elaborado por: Gerente de Operações Aprovado por: Diretoria

1. OBJETIVO

Padronizar a montagem dos packs de bateria, garantindo a correta conexão em série das células, o torque adequado dos barramentos (busbars) e a instalação dos sensores de temperatura, assegurando a performance e a segurança do conjunto.

2. ESCOPO

Este procedimento aplica-se a todos os grupos de células aprovados no POP-002. É executado pelos Técnicos de Montagem.

3. RECURSOS NECESSÁRIOS

  • Ferramentas: Torquímetro com isolamento VDE 1000V, soquetes adequados.
  • Componentes: Grupo de 16 células (aprovadas), barramentos de cobre, parafusos, arruelas, sensores de temperatura, isoladores.
  • EPIs: Luvas isolantes Classe 00, óculos de proteção, calçado de segurança.
  • Documentação: Ordem de Produção (OP).

4. PROCEDIMENTO

  1. Preparação: Posicionar as 16 células aprovadas no gabarito de montagem, seguindo a orientação de polaridade (+/-) definida no projeto.
  2. Limpeza: Limpar os terminais das células e os barramentos com álcool isopropílico para remover qualquer resíduo.
  3. Instalação dos Sensores: Fixar os sensores de temperatura nos pontos indicados no projeto (geralmente nas células centrais do pack).
  4. Conexão em Série: Posicionar os barramentos de cobre, conectando o terminal positivo de uma célula ao negativo da célula seguinte, até que todas as 16 células estejam em série.
  5. Torque dos Parafusos: Utilizando o torquímetro calibrado (conforme IT-002), apertar todos os parafusos com o torque especificado de 8 Nm ±5%. O torque inadequado é uma falha crítica que pode causar superaquecimento.
  6. Inspeção Visual: Verificar se todos os barramentos estão corretamente posicionados e se não há objetos estranhos entre as células.

5. REGISTROS

  • Apontamento na Ordem de Produção (OP).

6. SEGURANÇA

ATENÇÃO:

  • O pack de bateria agora possui uma tensão perigosa (~51.2V). O uso de EPIs é obrigatório.
  • Nunca utilize ferramentas não isoladas.
  • Não cause curto-circuito entre os terminais principais do pack.
