1. OBJETIVO

Detalhar o procedimento de conexão e programação dos parâmetros do BMS Daly utilizando o software "PC-Master", garantindo que todas as proteções e funcionalidades do BMS estejam configuradas corretamente para cada modelo de bateria.

2. EQUIPAMENTOS E SOFTWARE

  • BMS: Daly Smart BMS (modelo específico para o pack)
  • Software: Daly PC-Master V3.2
  • Interface: Adaptador USB-RS485/CAN
  • Computador: Estação de trabalho com Windows 10/11

3. PROCEDIMENTO PASSO A PASSO

3.1. Conexão Física

  1. Após a instalação do BMS no pack (conforme POP-004), conecte o adaptador USB-RS485/CAN à porta de comunicação do BMS.
  2. Conecte a outra extremidade do adaptador (USB) ao computador.
  3. O Windows deve reconhecer o dispositivo e atribuir uma porta COM (ex: COM3). Verifique no "Gerenciador de Dispositivos" qual porta foi atribuída.

3.2. Conexão do Software

  1. Abra o software PC-Master.
  2. Na tela inicial, selecione a porta COM correta no menu suspenso "Port".
  3. Clique em "Connect". O software tentará se comunicar com o BMS.
  4. Se a conexão for bem-sucedida, o status mudará para "Connected" e os campos da tela "Basic Info" serão preenchidos com os dados atuais do BMS.

3.3. Carregando e Verificando Parâmetros

  1. Vá para a aba "Parameter Setting".
  2. Clique em "Load Parameters from File".
  3. Navegue até a pasta de configurações e selecione o arquivo de configuração correspondente ao modelo da bateria em montagem (ex: SelEnergy_48V_230Ah_Config.cfg).
  4. Os parâmetros do arquivo serão carregados nas caixas de texto da tela. NÃO clique em "Write" ainda.
  5. Verifique visualmente os parâmetros críticos carregados, comparando-os com a tabela de referência abaixo.

3.4. Tabela de Parâmetros de Referência (Exemplo: 48V 230Ah)

3.5. Escrevendo os Parâmetros no BMS

  1. Após a verificação, clique no botão "Write Parameters to BMS".
  2. Uma barra de progresso aparecerá. Aguarde a conclusão.
  3. Ao final, uma mensagem de "Write Successful" será exibida.

3.6. Verificação Final

  1. Clique no botão "Read Parameters from BMS".
  2. O software irá ler os parâmetros que acabaram de ser gravados.
  3. Compare novamente os parâmetros lidos com a tabela de referência para garantir que a gravação foi 100% correta.
  4. Vá para a aba "Real-time Data" e verifique se as tensões individuais de cada uma das 16 células estão sendo exibidas corretamente.
  5. Desconecte o software clicando em "Disconnect" e remova o adaptador USB.

4. SOLUÇÃO DE PROBLEMAS

  • "Cannot connect to BMS":
    • Verifique se a porta COM selecionada está correta.
    • Verifique se o chicote de balanceamento está corretamente conectado ao BMS.
    • Certifique-se de que o pack de bateria tem carga suficiente (pelo menos 20% de SoC).
  • "Write Failed":
    • Tente novamente. Se o erro persistir, reinicie o BMS (desconectando e reconectando o chicote de balanceamento) e tente novamente.
    • Se o problema continuar, substitua a placa do BMS e abra um RNC (POP-010).
